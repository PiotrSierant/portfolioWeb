### Installation

_Below is an example of how you can  installing app._

1. Clone the repo
   ```sh
   git clone https://github.com/PiotrSierant/portfolioWeb.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start project
   ```sh
   npm start
   ```
